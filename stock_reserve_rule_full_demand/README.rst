This module allows to define a 'Full Demand' removal rule in stock
reservation rules.

**Table of contents**

.. contents::
   :local:

Use Cases / Context
===================

When having several picking zones, some have higher picking costs. When
the zone has not the full desired product quantity, we should not
reserve those products.

Configuration
=============

- Go to Inventory > Configuration > Warehouse Management > Reservation
  Rules
- On the desired rule for the desired location, use the 'Full Demand'
  removal strategy
- If you want to check first if product is not available elsewhere, fill
  in the 'Full Demand Fallback Locations' field.
